Getting Started

Prerequisites

The following are required to run BOM Weather:

  • An internet connection for downloading weather data
  • The Java JDK is required to compile and run BOM Weather. Link:
    Java download

How to Run

Run command prompt/terminal depending on OS (or other preferred shell application). Navigate to the folder named Compiled Distribution and run the following command:

java bom.jar

Compiling with Eclipse

To compile the program in eclipse, import the program into the file explorer by entering File -> Import. Then select Existing projects and navigate to the root directory of the program (Source Code). The program should be in the workspace and you can compile by pressing F5.
