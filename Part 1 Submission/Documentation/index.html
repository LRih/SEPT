Getting Started

Prerequisites

The following are required to run BOM Weather:

  • An internet connection for downloading weather data
  • The Java JDK is required to compile and run BOM Weather. Link:
    Java download

How to Run

Run command prompt/terminal depending on OS (or other preferred shell application). Navigate to the folder named Compiled Distribution and run the following command:

java bom.jar
